# 🤖 AI Chatbot Project

An AI-powered chatbot built using Python that provides intelligent, real-time responses using Natural Language Processing techniques and a simple web interface.

---

## 📌 Project Overview

This chatbot application allows users to:

- 💬 Interact in real-time
- 📖 Retrieve responses from a knowledge base
- 🌐 Use a simple HTML frontend interface
- ⚡ Get fast and efficient responses

---

## 🧠 Technologies Used

- Python
- HTML / CSS
- NLP Concepts

## 🚀 How to Run

1. Clone repository
2. Create virtual environment
3. Install requirements
4. Run `python app.py`
